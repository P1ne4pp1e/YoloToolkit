# 项目记忆

## Python 运行环境

- 本项目的所有 Python 脚本必须在名为 `YOLO` 的 conda 环境中运行。
- 文档、批处理脚本和命令示例必须先激活 `YOLO` 环境，不能使用系统 Python 或其他 conda 环境。

## 长时间任务进度

- 处理大规模数据集、图像集、训练、批量推理或其他耗时步骤时，必须在控制台显示实时进度条、已处理数量、总数量、速度和预计剩余时间。
- 默认使用 `tqdm`；多阶段或并行任务需要同时展示多个进度时，使用 `rich.progress`。
- 进度条应包裹实际处理循环，不能只在开始或结束时输出静态文本。

## 批处理脚本

- 用户要求生成 `.bat` 脚本时，必须将 conda 环境名、输入目录、输出目录、阈值等运行配置直接写在脚本顶部。
- 批处理脚本应支持双击直接运行；除非用户明确要求，否则不能依赖命令行参数传入这些配置。
- 需要输出中文的 `.bat` 脚本必须设置 UTF-8 控制台代码页和 `PYTHONIOENCODING=utf-8`，避免 `cmd.exe` 中文乱码。
